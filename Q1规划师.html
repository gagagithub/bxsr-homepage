Q2 规划师个人决战指令

INDIVIDUAL WAR DIRECTIVES · 2026 Q2

江宁
# 转化提速·存量深耕 #
工作重点 / Key Focus
  • ● 转化率突破：尽快将第一季度的讲解转化率优化至 70% 以上平均水平。
  • ● 存量资源活化：针对 Q1 留存 of 103 个已沟通客户逐一进行二次深度开发与补讲。
管理注意点 / Attention

Q2 必须全力闭合“临门一脚”的开发缺口，确保高质量客户不因后端转化无力而流失。

林付贤
# 前端突围·极致触达 #
工作重点 / Key Focus
  • ● 强化获信能力：针对添加率（61%）与沟通率（10%）全团最低的短板，进行专项攻坚。
  • ● 扩大漏斗基数：适度加强前端互动频率，优化初次触达话术，让顶级收口能力覆盖更多流量。
管理注意点 / Attention

保持王牌收割效率的同时，必须在获客前端投入更多精力，将“窄漏斗”变为“宽漏斗”，实现规模化稳定盈利。

马康挺
# 港险专家·经验分享 #
工作重点 / Key Focus
  • ● 流量加权发放：Q2 将显著增加线索分发量，充分释放专业战力。
  • ● 核心险种攻坚：集中处理 IUL、融资保单等高专业度香港保险新型需求。
  • ● 团队赋能分享：针对新型需求的客户处理案例，每周进行全团经验萃取分享。
管理注意点 / Attention

充分发挥香港保险从业经验丰富的核心特点。作为团队“技术担当”，通过实战经验分享提升全员对新型、高产值保单的处理上限。
