一、核心结论（一页速读）

  1. 本周激活率双双跃升。林付贤 19% → 32%（连续三周向上：4 月 2% → 5 月 19% → 本周 32%），马康挺 11% → 24%（突破连续两周低位）。
  2. 共同改进点：开场不再是"在吗""你好"。本周 14 个激活客户里，12 个开场带"上次/之前/你说"等个人化记忆点（vs 江宁分析里付贤历史 1.5%、康挺 10.6%），这是结构性突破。
  3. 付贤的两条主路：个人化记忆点 + 时效钩子（华哥、多才多艺、夏导）+ 服务请求转销售（hy燕子的差旅报销 → 顺势讲利率下调）。
  4. 康挺的两条主路：个人化记忆点 + 新产品名（潘总"安盛盛利 2 年交"、金女士"你说想做代理"）+ 客户拒绝后持续追问（仁钦卓玛 / 陈女士两段反弹）。
  5. 两人共同短板：方案制作率 / 讲解率 0%。激活做到了，但漏斗下游"约电话 / 讲解方案 / 邀约见面"几乎没跑通。这是下周必须突破的卡点。
  6. 付贤特有问题：5 月 1 日"五一假期"群发后遗症未清。本周 8 个激活里 3 个客户的"上次对话"都是同一条群发文案，激活回复明显冷淡（[抱拳]、"你想说什么意思"、"对"）。
  7. 康挺特有问题："信息倾倒"倾向，岁月静好案例一次连发 200+ 字的产品介绍，命中江宁分析里的失败模式。

三、林付贤本周做对了什么（4 大打法）

打法 1：开场带"上次产品类型 + 新产品/时效钩子"

这是付贤本周用得最多的开场结构（5/8 案例），与江宁打法 2 完全同款。

案例 A：华哥（沉默 12 天后激活）

付贤开场连发：

"华哥，在不，之前我们沟通过香港储蓄险的"
"图中的保险公司有没有比较熟悉的。"
"现在也有一些新产品上线啦。"

客户回"你想说什么意思"略冷，付贤承接："上面我整理出来的关于保险公司的介绍，你自己有没有比较看好哪家公司的产品" + 立刻给具体新品："现在有一款周大福的产品和太平洋的新品上线，两款产品都是非常顶尖的，之前你这边也有了解过的，要不要近期了解一下？" 客户立刻松口"可以呀"。

钩子拆解：① 历史关系（"之前沟通过"）② 具体品类（"香港储蓄险"）③ 新产品时效 ④ 客户冷反应后用"反问 + 具体品牌"二段强化。
案例 B：多才多艺铅笔（沉默 0 天但断点）

付贤开场：

"下个月，也就是 6 月底，国内的产品进一步会迎来定价利率的下调喔。你这边准备上车不。"

客户立刻"可以阿，说说利息"，付贤反问年龄 + 一次性预算金额，迅速进入方案演示。

时效钩子三件套：① 具体到月份（"6 月底"）② 政策动作（"定价利率下调"）③ 直接行动诱导（"准备上车不"），高效实用。
▎ 要点
"新产品 / 利率变化"是付贤做得最稳的钩子。要继续保持，每周开场前先看一眼"本周有什么新动态"作为统一弹药。

打法 2：客户拒绝后的二段反弹（不死缠也不放手）

这是江宁打法 4 的复刻，付贤本周开始稳定使用。

案例：大爱无疆（客户主动拒绝）

客户开口："下午好" + "近段时间不想考虑港险了"。付贤反弹：

"啊？确定不考虑了么"
"是对哪方面不太满意呢"

客户回"说不来，暂时不考虑"。付贤继续："恩恩，好的，明白。家里人不太同意是吧。" + 留接口："也可以一步一步先了解的，等后期想要考虑了就可以直接上车的。现在毕竟配置香港储蓄险也是一个趋势的哈"。

三步：① 先反问"确定吗" ② 追问"哪方面不满意" ③ 不死缠 + 留接口 + 给个软理由（"配置香港储蓄险是趋势"）。

打法 3：服务请求转销售

江宁打法 3 的应用，付贤抓得很准。

案例：hy燕子（差旅报销 → 香港卡 + 利率下调）

客户来要钱（差旅 3200 报销）。付贤先回应服务："3200 我都提交上去的哈，放心的"。然后顺势：

"我这两天催一下哈，到了我直接转给你，你现在把你香港银行卡发我，今天不到的话，下周也会到。我直接转你"

客户问"近期有去香港吗"，付贤立刻把对话从"报销"切到"业务"：

"去的呢，最近非常忙，马上后面还会降利率。6.5% 要降到 6% 了"
两步切换：① 服务接住（钱到位） ② 借香港卡话题自然过渡到"利率下调"，把售后变成销售触点。

打法 4：保单到期通知 → 客户关心 → 关系延展

案例：朱老板（保单续期通知激活）

付贤发标准保单续期通知。客户回："行 我在等等 美元到 6.6 更好"。付贤抓住机会切到关心：

"朱老板，最近忙不忙呀"

客户："最近还行 做了个手术养着呢"。付贤立刻："咋做手术了？什么情况"。客户："哦 没事 小手术"。付贤："恩，那就行。多保重"。

业务通知 + 真诚关心，强化人格信任。这种"业务窗口顺便问候"的小动作能维护长期关系。

⚠ 同一周仍有的瑕疵：群发后遗症

▎ 警示
本周 8 个激活里有 3 个客户（区松邦、华哥、老虎）的"上次对话"都是 2026-04-30 13:50-13:51 那条群发："其实 5 天假对我们来说还是值得期待一下的..."。这条群发当时被发给 2000+ 客户。

三个激活客户的回复都明显冷淡（[抱拳]、"你想说什么意思"、"对"），是激活但不温暖。下周必须把这种"广播板"模板从对话历史清出去——所有触达全部 1对1 个性化，不要再发统一文案。

四、马康挺本周做对了什么（4 大打法）

打法 1：开场带"客户上次说过的事 + 软追问"

康挺历史短板是"开场过短（首句平均 7.9 字）"，本周这个问题大幅改善。

案例 A：金女士（"你说想做代理"）

康挺开场：

"金女士 下午好，你说想做代理，但后面一直没有深入沟通咨询下产品呢。还有考虑合作吗？"
"需要在发些产品资料给你看看吗？"

客户简短回"忙一下下"，康挺持续追问，没有客套结束：

"是最近比较忙吗？看看是否在沟通下产品计划呢"
"那边有客户咨询合作吗？"

客户回"我做"，康挺接"你是自己买保险？考虑香港的？"

三步：① 用"你说想做代理"精准记忆点切入 ② 客户冷反应不放弃 ③ 用具体二选一反问追到真实意向。
案例 B：潘总（"安盛盛利 2 年交"具体新品）

康挺开场：

"潘总 你好，你港险后来配置了吗？安盛盛利 2 年交新产品出来有看到吗？"
"如没有配置的话，看看是否沟通下的"

客户发语音回应。

钩子三件套：① 关系标签（"潘总"）② 历史问题（"港险后来配置了吗"）③ 具体新品名（"安盛盛利 2 年交"），全部精准。

打法 2：客户拒绝后的两段反弹

案例：陈女士（家庭事 + 拒绝 → 切到中意产品 → 切到香港）

康挺开场："陈女士 你好 你家里事有处理好吗？关于保险理财规划看看是否要在深入沟通下的"。客户："你好！暂时不需要了，谢谢你！"

康挺第一段反弹（不放手 + 留接口）：

"我们保险涉及很全呢，后期不管哪种保险考虑，都可以帮你测评比较的"
"欢迎随时在来聊聊[抱拳]"

客户主动透露："不好意思，我前段时间才买了中意的保险了" + "香港那边的保险保费是不是起投都很高？"

康挺第二段反弹（直接给数据 + 反问）：

"国内保险我们一直是主推且卖的很好呢，你考虑国内也可以给你分析的-中意 中英 陆家嘴等等呢"
"香港不高呢 5 年交 年交 1 万美金就好了"
"你中意买多少保费的啊？"

客户拒绝去香港"时间和金钱成本超出预算"。康挺保留余地："如去成本都是我们报销的" + "后面有考虑，可以再聊聊的"。

两段反弹的关键：被拒后不收工 → 留接口 → 客户透露真实情况 → 用数据反问 → 客户再拒后再留口。

打法 3：客户主动来询时的"专业内容输出"

案例：吴先生（主动咨询固收 vs 分红）

客户主动："你好，我准备配一部分固收，一部分分红型？请你把固收第一梯队名称发给我..."

康挺先澄清需求："你是要纯固收？不带分红的产品？"。客户："纯固收，和分红都有想法"。康挺给出专业判断 + 推荐：

"国内纯固收太低了，现在产品预定利率保监会要求都是 2% 以下的。分红产品起码保证 1.75% + 分红，稍微分点红就是远超纯固收了。我们不太推荐国内固收产品"
"你要是考虑固收，可以考虑香港的。"

紧接着给出两线对比：

"这是国内 2 家头部收益的分红增额寿：1：复星保证 1.75% + 分红 演示 4.25%；2：中意 保证 1.5% + 分红 演示 4.25%"
"这是国内 2 家热卖的快返分红年金：1：中意 保证 1.75% + 分红 演示 3.9%；2：太保 保证 1.75% + 分红 演示 3.75%"
先澄清需求 → 用专业判断纠偏（固收太低）→ 给具体产品 + 数字对比，是销售顾问该有的姿态。

打法 4：客户疑虑时的"概念澄清 + 替代方案"

案例：刘女士（杠杆寿误解 + 腿疾不能去香港）

客户后台留言。康挺主动："你看的是这篇文章对吧？" + "我们也做寿险，这款产品不是杠杆寿。这是养老年金"。

客户怀疑保司："大家人寿是内地小公司吧？"。康挺立刻给出公司背书：

"行业内很有名的，以前叫安邦保险，现在中国保险基金控股的"

客户提障碍："没法去香港，腿疾"。康挺立刻给替代："那聊聊国内保险也行呢，投保也方便"。

三步：① 澄清概念误解（"不是杠杆寿"）② 用品牌历史化解信任（"以前叫安邦保险"）③ 障碍立刻给替代（腿疾 → 国内保险）。

⚠ 同一案例的瑕疵：信息倾倒

▎ 警示
上面"刘女士"案例后半段，康挺一次连发了两条 200+ 字的产品介绍长文：
"六个维度展示硬实力：1️⃣ 高保底 1.75% 预定利率... 2️⃣ 利益高支持年金红利双交清增额... 3️⃣ 高现价现金价值..."

这是江宁分析里典型的"信息倾倒"失败模式。客户更愿意看条理清晰的"先 1 句结论 + 后续 2-3 个要点"，而不是一次塞 200 字。下周如果发产品介绍，建议拆成 3-4 条短句逐条发。

1. 方案制作率 / 讲解率 0% — 漏斗下游断层

▎ 关键问题
激活做到了（点燃对话），但漏斗下游"约电话 / 讲解方案 / 邀约见面"几乎没跑通。本周 14 个激活客户里，有清晰下一步行动召唤的只有 2-3 个（夏导邀约见面、多才多艺进入方案演示）。

建议：本周已激活的客户里，付贤至少挑 2 个、康挺至少挑 2 个，下周一开始主动追"约方案讲解电话"，把激活转成讲解。

2. 行动召唤过于模糊

本周两人都频繁出现"看看是否..."、"再聊聊..."、"随时联系" 这类无锚定的召唤。

模糊召唤建议改写
"看看是否还需要在沟通下的" "明天上午 10 点方便我打个电话给你 5 分钟介绍下吗？"
"随时可以在聊聊合作" "周三下午我把这款产品的方案做完发你，你看完我们约 15 分钟聊？"
"后期不管哪种保险考虑，都可以帮你测评" "你下周方便给我 10 分钟，我把中意 + 复星 + 太保三家放一起对比给你看？"
▎ 要点
召唤句必须带具体时间 + 具体动作 + 具体时长。"看看是否..."不算召唤。

3. 客户多问题时的回应节奏

付贤"老虎"案例：客户连发"有没有人民币固定年金保单？""有没有人民币固定储蓄险保单"，付贤间隔 20 多分钟才回，且只回了一句"又要人民币保单，又要固定的，这个是没有的"——客户回"谢谢"，对话冷却。

建议：客户连发多个问题时：① 当场回应一句先承接（"好的，我来给你一一对应"），不要让客户等待 20+ 分钟 ② 不要简单否定（"没有"），而是给替代方案（"人民币固定收益没有，但港币高保证 + 分红的产品您要不要看一下？"）。

六、给两人的下周可复制清单

林付贤 · 下周清单

  1. 继续保持"个人化记忆点 + 时效钩子"开场（华哥、多才多艺案例）。每周开工前先看一眼"本周有什么新动态/利率变化"作为统一弹药。
  2. 彻底清掉群发模板。本周 3 个激活客户的"上次对话"还是 4/30 那条 5 天假群发，下周开始全部 1对1 个性化触达。
  3. 把本周激活的 2 个客户推到方案讲解。多才多艺已经问"说说利息"、夏导主动来要资料，这两个最该追到"约电话讲解方案"。
  4. 客户多问题时不要让对话停留 20+ 分钟。当场回应一句承接，不要简单"没有"。
  5. 召唤句换成"具体时间 + 具体动作"。下周新发量翻倍但讲解 0，必须把承接质量跟上。

马康挺 · 下周清单

  1. 继续保持"客户上次说过的事 + 具体新品名"开场（金女士、潘总案例）。这是康挺历史最大短板（首句 7.9 字）的突破。
  2. "客户拒绝后两段反弹"打法已经稳定（陈女士、金女士），继续按这个节奏走，注意每段反弹之间留呼吸空间，不要连发太密。
  3. 改"信息倾倒"为"3-4 条短句拆解"。刘女士那种 200+ 字的产品介绍要拆成"先 1 句结论 + 后续 2-3 个要点"。
  4. 方案制作率从 12% → 0% 是退步。上周已经做到 12%，下周必须重新打开漏斗下游，把本周 17 个新发里至少 2 个推到"做方案"。
  5. 召唤句锚定时间。"看看是否再聊聊"换成"周三下午我把方案发你 + 约 15 分钟"。
▎ 团队层面建议
